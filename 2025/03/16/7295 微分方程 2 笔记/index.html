调和函数的船新方法

我们可以暴躁一点, 所有函数都是光滑的.

Harnack 不等式

Aim: 证明 Harnack 不等式

Harnack 不等式: \(\sup_{\Omega'} u \leq C\inf_{\Omega'} u\) 对于正调和函数 \(u\) 及 \(\Omega' \sqsubset \Omega\).

  • 传统证明: 积分, 平均值不等式, 对梯度再用平均值不等式.
  • 现在证明 (Nilenberg 等): 用极值原理.

Slogan: 只要对球做就好. 原因: 由于极大极小值 (能取到! 考虑 \(\bar\Omega'\) ??) 之间连线可被球有限覆盖.

只要有 (对数梯度估计) \(\sup_{B_{1 / 2}} \frac{|\partial u|}{u} \leq C\) 就好, (用 Newton-Leibnitz 公式).

怎么 推导对数梯度估计? 丘成桐 1975.

定理 (Yau). 若 \(u > 0 \in C^3(\Omega)\) 调和, 且 \(B_1 \sqsubset \Omega\), 那么 \(\sup_{B_{1 / 2}} |\partial\log u| \leq C_n\).

证明.

取 \(v = \log u\), 那么 \(u = \exp v\), \(u_i = v_i\exp v\), \(0 = \Delta u = (|\partial v|^2 + \Delta v)\exp v\), i.e. \(\Delta v = - |\partial v|^2\) (1, 方程及其一阶导).

取辅助函数 \(\varphi = (1 - |x|^2)^2 |\partial u|^2\), 由于 \(\varphi\) 在边界等于 \(0\), 且 \(\varphi \geq 0\), 那么可设 \(\varphi\) 在 \(B_1\) 内部 \(x_0\) 处取极大值, 下述在 \(x_0\) 处运算. 则此时 \(\partial\varphi(x_0) = 0\) (2), \(\Delta \varphi(x_0) \leq 0\) (3, 极值b). 通过 (1), (2), (3) 证明要证的 \(|\partial v| \leq C_n\).

取 \(\eta := 1 - |x|^2\), \(\varphi := \eta^2 |\partial v|^2\), \(0 = \varphi_i = 2\eta\eta_i |\partial v|^2 + \eta^2 2 |\partial v||\partial v|_i\), i.e. \(-2\eta_i|\partial v|^2 = \eta (|\partial v|^2)_i\) (2', 极值a).

再带入 (3): 即 \(0 \geq \Delta \varphi(x_0) = \Delta(\eta^2)|\partial v|^2 + 2(\eta^2)_i(|\partial v|^2)_i + \eta^2\Delta(|\partial v|^2)\).

  • 第一项: 低阶项, 留下.
  • 第二项: 用 (2') 处理, 如下
    • \(= -8\eta_i\eta_i |\partial v|^2\).
  • 第三项: 希望往 \(|\partial v|\) 方向化简. 具体计算如下
    • \(\Delta(|\partial v|^2) = (v_jv_j)_{ii} = 2v_{jii}v_j + 2v_{ji}v_{ji}\). 再用 \(v_{jii} = (\Delta v)_j\), 再用 (1), 得到 \(v_{jii} = - (|\partial v|^2)_j\).
    • 从而第三项 \(= 2\eta^2 (-|\partial v|^2_jv_j + v_{ij}v_{ij})\).
    • 由于 \(v_{ij}v_{ij} \geq \sum_i v_{ii}^2 \geq (\Delta v)^2 / n = |\partial v|^4 / n\). 最后是 Cauchy 不等式和 (1).
    • 从而大于 \(2\eta^2(|\partial v|^4 / n - |\partial v|_j^2v_j) = 2\eta^2|\partial v|^4 / n - 4\eta\eta_j|\partial v|^2v_j)\), 最后一步用了 (2').

于是 \(0 \geq \Delta(\eta^2)|\partial v|^2 -8\eta_i\eta_i |\partial v|^2 + 2\eta^2|\partial v|^4 / n - 4\eta\eta_j|\partial v|^2v_j)\). 同除 \(Bv\) (定义为 \(|\partial v|^2\)), 得到

\[0 \geq \Delta \eta^2 - 8B\eta + 2\eta^2 |\partial v|^2 / n - 4\eta\eta_jv_j.\]

用 Cauchy 不等式拆 \(4\eta\eta_i v_i\), 得到 \(\eta^2 Bv / n + 4n B\eta\), 带入 \(\eta_i = -2x_i\), \(B\eta = 4x^2\), \(\Delta \eta = -2n\), \(\Delta\eta^2 = 2\eta \Delta\eta + 2\eta_i\eta_i\). 于是带入, 并利用 \(\eta^2 < 1\), \(x^2 < 1\):

\[\begin{aligned}0 &\geq -4n\eta + 8x^2 - 32x^2 + \eta^2 Bv / n - 16nx^2\\ &\geq -4n - 24 -16n + \eta^2 Bv / n.\end{aligned}\]

即 \(20n + 24 \leq \eta^2 Bv / n\).

于是 \[\sup_{B_{1 / 2}} \eta^2 Bv / n \geq \sup_{B_{1 / 2}} Bv / n \inf_{B_{1 / 2}} \eta^2 \geq \frac{9}{16} \sup_{B_{1 / 2}} Bv / n.\]

于是再用 (1), 得到 \(\sup_{B_{1 / 2}} \Delta v \leq \frac{16}{9} n (20n + 24)\).

定理得证.

注记. 我们此处的思路就是使用不等式把问题往 \(\Delta v\), 即 \(Bv\) 方向化简. 再使用约束 (\(x < 1\) 什么的) 把不重要的量放成 \(1\).

最大值原理

\(\Delta u = 0\), \(u\in C^2 \implies u\in C^\infty\). 方法:

  1. 平均值, 恒等逼近.
  2. 高阶导数估计.

Harnack 不等式已经证明 (对数估计 \(\sup_{B_{R / 2}} |\partial\log u| \leq C_n / R\), Scaling, 如何理解? \(u(y) = U(Rx) = \tilde{u}(x)\), \(\partial \log \tilde u = \partial_y \log u(y) \partial_x y = R\partial \log u\)).

作业: \(\Delta u = 0 \implies \sup_{\bar \Omega} Bv \leq \sup_{\partial\Omega} Bv\), 整体梯度估计. 今天还介绍梯度内估计.

什么是估计? 有了估计才能用泛函分析得到存在性, 我们在估计时, 不知道是否有界. 这是一种先验估计. 对方程所有的解得到估计 (比如, \(W^{k, p}\), \(C^{k, a}\) \(\impliedby\) 有界, 梯度, Holder).

梯度内估计. 在 \(\Omega' \sqsubset \Omega\) 中的估计, 称为内估计, 这与边值 \(\partial\Omega\) 无关.

椭圆方程, 最重要的是内估计.

  • \(\Delta u = 0\), 有;
  • \(H(u) = 0\), 有 1969;
  • \(\det \nabla^2 u = 0\), 无 1971 (辅助函数被换了);
  • 二阶, 未知.

下面讨论内估计.

定理. \(\Delta u = 0\), 则 \(\sup_{B_{1 / 2}} |\partial u| \leq C_n / R \max_{\bar B_{1}} |u|\).

证明. 找辅助函数.

取 \(\eta = 1 - |x|^2\) (简单的时候可以用这个, 换掉能发文章...), 取 \(\varphi := \eta^2 Bu + \alpha u^2\) (\(\alpha\) 待定, Berstein 技巧, 1900s, 低阶项 (?) 帮忙)

如果 \(\varphi\) 在 \(\partial B_1\) 达到极值, 则结论成立: \[9 \sup_{B_{1 / 2}} Bu / 16 \leq \inf_{B_{1 / 2}} \eta^2 \sup_{B_{1 / 2}} Bv \leq \sup_{B_{1 / 2}} \varphi \leq \sup_{B_1} \varphi = \sup_{\partial B_1}\varphi = \alpha_0 \sup_{\partial B_1} Bu.\]

问题归结为, 如何找 \(\alpha\) 使得 \(\varphi\) 在边界达到极值.

来找 \(\alpha\).

性质: 回忆 Fermat 引理, \(u\) 在 \(x_0\) 达到最大 \(\implies\) \(\nabla u = 0, \Delta u \leq 0\). 故如果 \(\Delta \varphi > 0\), 那么 \(\varphi\) 在只能边界达到极大值, 实际上, \(\Delta \varphi \geq 0\) 足矣让极大值在边界取得. 令 \(v := \phi + \varepsilon x_1^2\), 则 \(\Delta \phi \geq 0\) 能推出 \(\Delta v > 0\), 再令 \(\varepsilon \to 0\) 即得.

下面阐述找 \(\alpha\) 的方法.

\(\varphi_i = \eta^2_{, i} Bu + \eta^2 Bu_i + \alpha u^2_{,i}\) (注意顺序! \(\eta_{,i}^2 := (\eta^2)_i\), \(Bu_i := (Bu)_i\), 因为 \(B\) 只对标量函数作用! 或者 \(B\) 的优先级高于导数).

\(0 \geq? \Delta \varphi = \Delta(\eta^2) Bu + 2\eta^2_{,i} Bu_{i} + \eta^2 \Delta Bu + \alpha \Delta(u^2)\) (注意顺序!).

带入 \(\Delta\eta^2 = 2\eta \Delta\eta + 4\eta_i\eta_i\). 并且 \(\Delta u = 0 \implies 2u\Delta u + 2Bu = 0 \implies u\Delta u = - Bu\).

  1. 保留,
  2. \(\eta^2_{,i} = 2\eta \eta_i\), \(Bu_i = 2u_ju_{ij}\), 所以为 \(-8 \eta_i \eta u_j u_{ij}\),
  3. \(\Delta Bv = 2\Delta u_iu_i = 2u_{ij}u_{ij}\), 所以为 \(2\eta^2 u_{ij}u_{ij} = 2\eta^2 |\partial^2 u|^2\),
  4. \(2u\Delta u + 4u_iu_i = 2Bu\).

化简得, \(2\eta^2 |\partial^2 u|^2 + 8\eta\eta_i u_i u_{ij} + [2\alpha + \Delta (\eta^2)] Bu \geq? 0\).

第一, 第三项都可以大于零, 想使得第二项大于零. 用 Cauchy 不等式.

  • \(8\eta\eta_i u_i u_{ij} \leq 2\eta^2B^2 u + 8 B\eta Bu\) (记 \(|\partial^2 u|^2\) 为 \(B^2 u\)).

从而 \(\Delta \varphi \geq 2\eta^2 |\partial^2 u|^2 - (2\eta^2B^2 u + 8 B\eta Bu) + [2\alpha + \Delta (\eta^2)] Bu = B^2u (2\eta^2 - 2\eta^2) + Bu(2\alpha + \Delta (\eta^2) - 8B\eta)\).

带入 \(\eta_i = -2x_i\), \(B\eta = 4x^2\), \(\Delta \eta = -2n\) 及 \[\Delta\eta^2 = 2\eta \Delta\eta + 2\eta_i\eta_i = -4n \eta + 4 x_ix_i.\] 得到

\[\Delta \varphi \geq Bu (2\alpha -4n\eta + 8x^2 - 32 x^2) = Bu (2\alpha - 4n + 4n x^2 - 24 x^2) \geq Bu(2\alpha - 4n -24),\] 所以取 \(2\alpha \geq 4n + 24\) 即可.

定理得证.

注记.

对 \(\Delta u + b_i u_i = 0\) 的小推广

下面均在有界区域中谈.

若 \(\Delta u \geq 0\), 则最大值在边界达到. 若 \(\Delta u \leq 0\), 则最小值在边界达到 (类似证明).

所以若 \(\Delta u = 0\), 则最大, 最小值均在边界达到.

换一个方程: \(\Delta u + b_i u_i \geq 0\) 呢?

此时, 最大值在边界达到.

证明.

  1. 若严格大于 \(0\), 那么直接分析得证只能在边界达到.
  2. 下面考虑辅助函数? 是否可以 (\(\Delta v + b_i v_i > 0\)?) 仍然用 \(v := u + \varepsilon x^2?\) 不行.

换成 \(v := u + \varepsilon e^{\beta x_1}\), 此时 \(\Delta v = \Delta u + \varepsilon \beta^2 e^{\beta x_1}\). 于是 \(\Delta v + b_i v_i = \Delta u + \varepsilon \beta^2 e^{\beta x_1} + b_i(v_i + \varepsilon \beta e^{\beta x_1}\delta_{i1})\). 取 \(\beta > 1 + |b|\) 即可.

得证.

作业:

  1. 取 \(v := u + \varepsilon e^{\beta x^2}\)?
  2. 找到 \(\alpha\) 的范围, 使得 \(\varphi := \frac{Bv}{v^\alpha}\) 在 \(\partial \Omega\) 达到最大值.

定理. 若 \(\Delta u + b_i u_i = 0\), 那么 \(\sup_{\bar \Omega} |\partial u| \leq c(1 + \sup_{\partial \Omega} |\partial u|)\).

证明. 取 \(\varphi := Bu + \alpha u^2 + e^{\beta x_1}\), 是否有 \(\Delta \varphi + b_i \varphi_i \geq 0\)? 计算

\[\begin{aligned} \Delta \varphi + b_i \varphi_i &= \Delta Bu + \alpha 2u \Delta u + 2 Bu + \beta^2 e^{\beta x_1} \\ & + b_i Bu_i + \alpha b_i u^2_{, i} + \beta e^{\beta x_1}b_1. \end{aligned}\]

一般二阶椭圆方程的先验估计

先验估计?

能否把内部梯度估计的前述的 \(\Delta\) 换成一个线性椭圆算子 \(L\)? 对数梯度估计呢? (这和边值无关)

关于 \(\Delta\) 边值问题 (Dirichlet / Neumann) 的先验估计 (有界) 是否能够推广到 \(L\) 呢? 如何做先验估计?

先验估计第一步是有界估计.

对数估计与 Harnack 不等式

利用三个结论:

  1. 方程及其一阶导数.
  2. \(\varphi_i(x_0) = 0\).
  3. \(\Delta \varphi(x_0) \leq 0\).

回忆 1 在化简第三项 \(\eta^2 \Delta Bv\) 时候用到.

用上述方法做一般的情况, 结论: \(\sup_{B_{1 / 2}} B\log u \leq C(n, |a_{ij}|_{C^1(\bar B_1)}, \lambda, \Lambda)\).

做一点线性代数, 若 \(A, B\) 为对称阵, \((a_{ij})_{ij}\) 为 \(A\) 的坐标表达.

  • 若 \(A \geq 0 \geq B\), 则 \(\operatorname{tr} AB \leq 0\).
  • 若 \(\lambda I \leq a_{ij}(x) \leq \Lambda I\), 则 \(a_{ij}^2 \leq a_{ii}a_{jj}\).

证明.

取 \(v := \log u\), 此时方程变为 \(a_{ij} v_{ij} = -a_{ij} v_iv_j\). 取 \(\varphi := \eta^2 w\), 其中 \(w := a_{ij}v_i v_j\), \(\eta\) 同上.

假设在 \(x_0\) 达到极大值, 所有计算在 \(x_0\) 处进行. 现在有:

  1. 方程及一阶导 (这是线性化算子出现的原因),
  2. \((\eta^2)_i w + \eta^2 w_i = \varphi_i(x_0) = 0\),
  3. \((\eta^2)_{ij} w + (\eta^2)_i w_j + (\eta^2)_j w_i + \eta^2 w_{ij} = \varphi_{ij}(x_0) \leq 0\), 这里指半负定.

Slogan: 不论是椭圆, 双曲还是抛物, 我们都必须用算子在 \(u\) 处沿 \(v\) 方向微分 (称为线性化算子) 的主部诱导的辅助函数. 此处线性化算子为 \(L_u (-) := a_{ij}(-)_{ij}\). (实际上 \(Bv\) 也可以)

将上述三者带入线性化算子 \(L_u \varphi = a_{ij} \varphi_{ij} = a_{ij}(\eta^2)_{ij} w + 2a_{ij}(\eta^2)_i w_j + a_{ij}\eta^2 w_{ij}\).

第二项, 使用 2 得到 \(2a_{ij} \eta^2 w_iw_j\).

第三项...

而由于 \(\varphi_{ij}(x_0) \leq 0 \leq a_{ij}\), 得到 \(L_u \varphi \leq 0\).
