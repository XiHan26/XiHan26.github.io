Riemann 流形好在哪里?

几何: 逐点局部全局性质的统一

回忆: 逐点-局部-整体的关系. Riemann 流形给出了这些关系之间的过渡.

  • 逐点信息: 切向量, 度量, 梯度, 体元.
  • 逐点到局部 (可能是 ODE): 指数映射 (\(\implies\) 测地凸邻域), 测地线, 度量, 变分.
  • 局部信息: 等距映射, 联络, 度量连续性, 曲率, 测地线, 测度.
  • 局部到整体: 全局测地线 (\(\implies\) 完备性, 指标, 割迹), 变分.
  • 整体信息 (拓扑信息): 基本群, 覆叠, (曲率) 整体条件.

我们下面一一介绍.

逐点性质

逐点层面的事情是关于二次型的线性代数, 我们不讨论. 但是值得提出, 选择 \(\partial_i\) 是和逐点绑定的. 一般来说, 局部上做计算更多使用移动标架 (向量丛的截面). 当然, 移动标架比较抽象, 而对于 Riemann 流形我们可以选择法邻域, 回到舒适的 \(\partial_i\) 境地.

逐点到局部

我们先介绍一点从逐点到局部的东西. 这种讨论的基石是度量连续.

Koszul: Levi-Civita 联络

但由于一些奇妙1的事情, 我们 (不需要度量连续) 有:

定理 (Koszul). 给定 Riemann 度量, 存在唯一与之适配的无挠 Levi-Civita 联络, 其 Cristofel 记号为 \[\Gamma_{ij}^k = g^{kl}\frac{1}{2}\left(\partial_i g_{jl} + \partial_j g_{il} - \partial_{l}g_{ij}\right).\]

通过这种适配关系, 我们将度量划归成了 Levi-Civita 联络.

积分与度量

积分是一种从逐点到局部 (或整体) 的方法. 由于欧氏空间 (通过积分) 逐点体积和局部体积相重合, 因此我们构造流形上的测度 \(\int \sqrt{G} \prod\mathrm{d} x^i\), 特别的, 用传统 P.O.U. 能进一步在 \(M\) 上积分.

  • 我们能够照搬欧氏空间的定理或照搬证明, 比如散度定理 (P.O.U.), 余面积公式 (证明);
  • 从测度出发, 我们可以定义 Riemann 流形上的度量, 我们可以问这个度量什么时候完备, 请看 Hopf-Rinow 定理 [[#^4a5b16]].
  • 我们还可以定义 Riemann 流形上的 Laplacian.
    • 值得注意二阶微分的性质, 这就是 Lecture 4 定理 2.8., 和 Laplacian 交换 \(\iff\) 是局部等距.
ODE 与变分

逐点到局部的方法也可以是 ODE (回忆 Cauchy-Lipschitz 定理). 请选择某个方程:

  • 测地线方程: \(\nabla_{\dot \gamma}\dot\gamma = 0 \iff \ddot{x}^k + \dot{x}^i\dot{x}^j\Gamma_{ij}^k = 0\);
  • 平行移动方程: \(\nabla_{\dot\gamma} X = 0\);
  • Jacobi 场方程: \(\nabla_{\dot\gamma}\nabla_{\dot\gamma} X + R({\dot\gamma}, X){\dot\gamma} = 0 \iff \ddot X^i e_i + X^jR_{1i1}^je_j = 0\);
  • Euler-Lagrange 方程 (变分).

巧的是, 它们都和积分的变分有关系, 测地线是能量的第一变分 \(\partial_s F(\gamma_s) = 0\), Jacobi 场是能量的第二变分 \(\partial_s^2 F(\gamma_s) = 0\). 变分的精髓在于 \(\nabla_{f_t}f_s = \nabla_{f_s}f_t\), 即 \([f_s, f_t] = 0\), 这又来自于 \([\partial_s, \partial_t] = 0\).

Jacobi 场

详细说说 Jacobi 场: 测地线 \(\gamma\) 上的 Jacobi 场是表征着如何对测地线做微扰使其还是测地的方法. 这是 Lecture 13 定理 1.7.. 证明是传统但是复杂的构造. 值得注意, Jacobi 场不能给出唯一的微扰测地线族, 就像切向量不能唯一给出曲线一样. Jacobi 场的维数是 \(2\dim M\).

指标形式

关于第二变分, 我们还可以谈论它的极化, 就是指标形式 \(I(X, Y)\) 或记作 \((X|I|Y)\).

指数映射

除此之外, 我们还会使用指数映射关联切空间和某个邻域.

定义 (指数映射). '将切向量变成流形上的点.'

注记. 不一定对 \(T_pM\) 都有定义. 若存在 (见 Hopf-Rinow 定理 [[#^4a5b16]]), 则意味着 \(M\) 完备.

特别的, 指数映射的切映射是 \(\operatorname{Id}\), 从而给出局部同胚. 进一步 Gauss 引理 告诉我们它实际上是径向等距.

局部性质

法邻域

下面详细说说法邻域:

定义. 法邻域是 \((U, x^i)\) 使得

  • \(g_{ij}(p) = \delta_{ij}\);
  • \(\Gamma_{ij}^k(p) = 0\);
  • \(\partial_kg_{ij}(p) = 0\).

的坐标邻域.

因为法邻域中的信息包含局部性 (度量的导数), 因此不奇怪的是, 其证明需要用到逐点到局部构造中的指数映射.

曲率, 联络与度量的关系

利用法邻域我们能够回答下述问题: 曲率是联络的 '微分' (想想 \(\partial_i\) 下的曲率表达), 那么能否用曲率还原联络? 实际上, 我们有这些定理:

  • 定理. \(g_{ij}(x) = \delta_{ij} + \frac{1}{3} R_{ijkl}(p)x^kx^l + O(|x|^3)\).
  • 定理. \(\det g_{ij} \to 1 - \operatorname{Ric}_{kl}(p)x^kx^l/3\).
  • 定理. \(\frac{2\pi r - L_r}{r^3} \to \pi K(\Pi_p) / 3\).
  • 定理. \(\operatorname{Area}(S_r(p)) \to m\omega_m r^{m - 1} - \frac{S(p)}{6}\omega_m r^{m + 1}\).

这是比较定理的基础.

曲率和度量还有以下的 Cartan 局部等距定理: 曲率相同 \(\implies\) 局部同胚. 正是因为有指数映射, 我们才能够说清楚它 (局部到局部. 但是由于鉴别曲率是逐点的, 证明需要指数映射).

值得注意, Cartan 局部等距无法减弱到截面曲率或 Ricci 曲率, 但是当常曲率时可以, 因此用 Cartan 局部等距可直接证明:

定理 (Riemann). 若 \(M\) 常截面曲率, 则任意一点有开邻域和空间形式的开集等距同构. ^27f07a

我们能够看到它的整体版本: Killing-Hopf 定理 [[#^75da75]]. 这也对比较定理的建立有贡献.

局部到整体

Slogan: 局部度量性质 \(\implies\) 拓扑性质.

下面介绍一些从局部到整体时会出现的障碍.

测地线局部最短 vs 全局最短

由于测地线总是在法邻域内最短的 (指数映射 + Gauss 定理). 我们一直想问: 什么时候测地线全局最短? 其回答分为

完备流形与 Hopf-Rinow 定理

首先, 第一个问题: 什么时候最短测地线存在? 回忆 Hopf-Rinow 定理: 若 \(M\) 连通 Riemann 流形, 则 TFAE ^4a5b16

  • \(M\) 是完备度量空间;
  • \(M\) 测地完备;
  • 对所有点, \(\exp_p\) 都在 \(T_p M\) 上有定义.
  • 存在 \(p\in M\), 使得 \(\exp_p\) 在 \(T_p M\) 上有定义.
  • 有界闭集紧.

它们都能推出 \(M\) 中任意两点能够被一条最短测地线连接, 并且 (根据定义) 这条测地线的长度就是这两点的距离.

有了完备流形的概念, 我们能够讨论指数映射什么时候是全局微分同胚. 回答是 Cartan-Hadamard 定理: 对于完备非负截面曲率流形 (Cartan-Hadamard 流形), \(\exp\) 是覆叠映射, 进一步地, 如果 \(M\) 是单连通的, 那么 \(M\) 和 \(T_pM\) 微分同胚. 这是比较定理最 naïve 的版本. 其证明就是抄一遍 Ambrose 定理2: 若 \(M\) 完备, 则 \(f\) 是光滑覆叠, 且 \(N\) 完备. ^2a4e24

共轭点与割点

第二个问题: 什么时候测地线总是最短?

第三个问题: 什么时候最短测地线唯一?

其回答分别是无共轭点和割点. 我们先把定义列下来.

  • 定义 (共轭点). 取定测地线 \(\gamma[a, b]\), 若存在 Jacobi 场在 \(t_1, t_2\in (a, b)\) 上均消失, 那么称 \(\gamma(t_1)\) 与 \(\gamma(t_2)\) 共轭.
  • 定义 (割点). 取定测地线 \(\gamma[0, b]\), 对于其上一点 \(p = \gamma(0)\), 定义 \(t_0 := \sup\{t|\gamma[0, t]\text{是最短测地线}\}\), 称 \(\gamma(t_0)\) 为 \(p\) 的割点.

然后举一些例子 (我们会在下面详细说说常曲率流形):

  • 例 (双曲度量). \(\mathbb{H}^n\)
    • 共轭点：无
    • 割点：无
  • 例 (球面). \(\mathbb{S}^n\)
    • 共轭点: 对径点
    • (第一) 割点: 对径点
  • 例 (柱面). \(\mathbb{S}^1\times \mathbb{R}\)
    • 共轭点: 无
    • 割点: 对径直线

所以共轭点和割点并不总一样. 进一步的, 非负截面曲率3流形没有共轭点, 但是可能有割点.

实际上, 我们有定理说明共轭点和割点的关系 (伍鸿熙, 黎曼几何初步, P157, 引理 4.). 草率来说, 区别就是全局最短和局部最短的关系.

回到标题: 局部到整体. 我们能够看到紧致和割迹的关系 (伍鸿熙, P159-161):

定理. \(M\) 完备流形, \(M\) 紧致等价于任取 \(p\in M\) 的割迹在切空间的原像中同胚于单位球面. 证明是使用 \(M\) 的直径且射线和割迹交点唯一.

除此之外, 进一步的, 我们有 \(\exp_x\) 在割迹内部是单的, 并且任意紧致流形拓扑上都是某个球面关于其边界的粘合.

这引出了球面定理.

我们自然还可以问, 什么时候 \(\exp\) 是覆叠. Cartan-Hadamard 定理 [[#^2a4e24]] 给了一个很好的回答, 其证明实际上可以推广到: 若某点 \(p\) 没有共轭点 (称为 pole), 那么其为覆叠.

整体性质

常曲率流形

常曲率流形对从局部过渡到整体有巨大贡献. 我们先从 Schur 定理开始, 它预示了曲率不可能是逐点量.

常截面曲率流形 (空间形式)

定义. 空间形式为 \(\mathbb{R}^n, \mathbb{H}^n\) 和 \(\mathbb{S}^n\) 带上标准度量.

回忆 Riemann 定理 [[#^27f07a]] 和 Killing-Hopf 定理, 后者是从局部度量性质给出整体性质 (拓扑限制) 的典范案例:

定理 (Killing-Hopf). 若 \(M\) 常截面曲率且完备, 则 \(M\) 的万有覆叠是空间形式. ^75da75

证明. 首先先排开九文大钱: 我们有 Cartan 局部等距和 Ambrose 定理. 接下来, 怎么使用它们?

实际上, 我们已经见过了所有的完备偶数维正常截面曲率流形.

常 Ricci 曲率流形 (Einstein 流形)

此时 \(\operatorname{Ric} = Sg / m\), \(S\) 为标量曲率.

由于 Chern-Gauss-Bonnet 定理, 拓扑限制如下:

定理 (Berger). 若 \(M\) 是可定向的闭四维流形, 那么 \(M\) Einstein 意味着欧拉示性数大于零.

这被 Hitchin-Thorpe 加强了.
